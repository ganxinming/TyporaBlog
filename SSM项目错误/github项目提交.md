1.先创建一个git仓库，然后分享到github



git命令复习：

git init:初始化一个git目录

git status:查看状态

git log:查看日志

git add 文件名  :提交到暂存区

git commit -m "" ，提交到本地仓库（即本地，相当于已经确认修改）

git push origin master  提交 （ 提交到github）

git push -f origin master 忽略版本错误，强制提交。
